# aetherfall-campaign-assets
Campaign Assets for Aetherfall Module

This repository contains visual, printable, and module-ready assets for the Daear / Aetherfall campaign setting.

Assets include:
- Homebrewery-ready images
- Cover art
- Chapter banners
- Character portraits
- Creature art
- Settlement and regional maps
- STL files for miniatures, terrain, and props
- Module handouts and visual references

The `00-module-ready` folder contains assets intended for direct use in the published Homebrewery module.
Other folders contain working files, drafts, alternate versions, and archival material.
